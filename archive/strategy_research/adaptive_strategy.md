# 自适应交易策略研究报告

## 摘要

本研究开发了一套能够根据市场状态自动调整的交易策略系统，不依赖固定参数。系统通过实时识别市场状态（趋势市、震荡市、突破市、极端市），动态选择最优策略，并自适应调整风险参数。

## 1. 市场状态识别模型

### 1.1 市场分类定义

| 市场类型 | 特征 | 识别指标 |
|---------|------|---------|
| **趋势市** | 价格持续单向运动 | ADX > 25, 均线斜率显著 |
| **震荡市** | 价格在区间内波动 | ADX < 20, 波动率低分位数 |
| **突破市** | 大幅波动伴随成交量 | 波动率 > 90 分位数，成交量放大 |
| **极端市** | 黑天鹅事件 | 波动率 > 99 分位数，价格跳空 |

### 1.2 核心识别指标

#### ADX (Average Directional Index)
- 计算 14 周期 DI+ 和 DI-
- ADX = 平滑后的 (DI+ - DI-) 绝对值
- 阈值：>25 趋势市，<20 震荡市

#### 波动率分位数
- 计算滚动窗口 (20 日) 真实波动率
- 当前波动率在历史分布中的分位数
- 阈值：<30% 低波动，>70% 高波动，>90% 极端

#### 价格分布形态
- 偏度 (Skewness): 衡量分布不对称性
- 峰度 (Kurtosis): 衡量尾部风险
- 极端市特征：峰度 > 5

#### 成交量特征
- 成交量比率 = 当前成交量 / 20 日平均成交量
- 突破市确认：比率 > 2.0

### 1.3 状态转换模型

采用隐马尔可夫模型 (HMM) 建模状态转换：

```
状态转移矩阵 P:
          趋势    震荡    突破    极端
趋势    0.7     0.2     0.08    0.02
震荡    0.15    0.75    0.08    0.02
突破    0.1     0.1     0.7     0.1
极端    0.05    0.05    0.1     0.8
```

## 2. 策略库设计

### 2.1 趋势策略

#### 均线交叉策略
- 快线：EMA(12), 慢线：EMA(26)
- 金叉做多，死叉做空
- 适用：强趋势市 (ADX > 30)

#### 通道突破策略
- 计算 20 日高低点通道
- 突破上轨做多，突破下轨做空
- 止损：通道中线

#### 动量跟随策略
- 计算 10 日收益率排名
- 做多最强品种，做空最弱品种
- 调仓周期：5 日

### 2.2 震荡策略

#### 均值回归策略
- 计算 20 日移动平均
- 价格 > 均值 + 2σ → 做空
- 价格 < 均值 - 2σ → 做多
- 目标：回归均值

#### 布林带策略
- 中轨：SMA(20)
- 上轨/下轨：±2 标准差
- 触下轨做多，触上轨做空
- 止损：布林带外 1σ

#### RSI 超买超卖
- RSI(14) < 30 → 做多
- RSI(14) > 70 → 做空
- 平仓：RSI 回归 50

### 2.3 突破策略

#### 波动率突破
- 计算 ATR(14)
- 买入突破：前日收盘 + 2×ATR
- 卖出突破：前日收盘 - 2×ATR

#### 成交量突破
- 成交量 > 3×20 日均量
- 价格突破 20 日高低点
- 确认信号：两者同时触发

### 2.4 对冲策略

#### 多空对冲
- 做多强势品种，做空弱势品种
- 市场中性，赚取相对收益
- 适用：趋势不明朗时

#### 跨品种对冲
- 相关性高的品种对 (如 BTC/ETH)
- 价差交易，均值回归

## 4. 风险控制自适应

### 4.1 波动率调整仓位

```python
def adjust_position(base_position, current_vol, benchmark_vol=0.02):
    """
    根据波动率动态调整仓位
    benchmark_vol: 基准波动率 (默认 2%)
    """
    vol_ratio = benchmark_vol / current_vol
    position = base_position * vol_ratio
    return clip(position, 0.1, 2.0)  # 限制在 0.1x-2x
```

### 4.2 相关性调整

```python
def correlation_adjustment(positions, correlation_matrix, threshold=0.7):
    """
    当多品种相关性上升时降低总仓位
    """
    avg_corr = mean(correlation_matrix)
    if avg_corr > threshold:
        reduction = (avg_corr - threshold) / (1 - threshold)
        return positions * (1 - reduction * 0.5)
    return positions
```

```python
def should_switch_strategy(current_strategy, alternatives):
    # 连续亏损检查
    if current_strategy.consecutive_losses >= 5:
        return True, 'pause'
    
    # Sharpe 检查
    if current_strategy.rolling_sharpe(20) < 0.3:
        return True, 'reduce_weight'
    
    # 最佳替代检查
    best_alt = max(alternatives, key=lambda s: s.rolling_sharpe(20))
    if best_alt.rolling_sharpe(20) > current_strategy.rolling_sharpe(20) + 0.5:
        return True, 'switch'
    
    return False, None
```
