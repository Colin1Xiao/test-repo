## ⚠️ 风险警示

加密货币合约交易是**极高风险**的投资行为：

- 🔴 **90% 以上的散户亏损**
- 🔴 高杠杆可能导致**本金全部损失**
- 🔴 极端行情下可能**穿仓**（亏损超过本金）
- 🔴 中国大陆用户需注意**合规风险**

**本指南仅用于教育目的，请谨慎交易！**

## 核心原则

### 1. 永不爆仓原则

```
爆仓价格 = 入场价 × (1 - 1/杠杆)  [做多]
爆仓价格 = 入场价 × (1 + 1/杠杆)  [做空]
```

**示例**：
- BTC 价格：$68,000
- 杠杆：50 倍
- 爆仓价格：$68,000 × (1 - 1/50) = $66,640
- **只需下跌 2% 就爆仓！**

### 2. 2% 规则

**单笔交易最大亏损不超过本金的 2%**

```
仓位大小 = (本金 × 2%) ÷ 止损%
```

**示例**：
- 本金：10,000 USDT
- 风险：2% = 200 USDT
- 止损：1.5%
- 仓位 = 200 ÷ 0.015 = **13,333 USDT**

### 3. 30% 规则

**总仓位不超过本金的 30%**

避免一次性满仓，保留充足保证金应对波动。

## 止损策略

### 止损类型

#### 1. 固定止损（推荐新手）

```bash
python3 scripts/calculate_stoploss.py \
  --entry 68500 \
  --stop-loss 2 \
  --take-profit 4
```

**输出**：
```
入场价格：68,500 USDT
止损价格：67,130 USDT (-2%)
止盈价格：71,240 USDT (+4%)
盈亏比：2:1 ✅
```

#### 2. 追踪止损（推荐）

自动跟随价格上涨，锁定利润。

```bash
python3 scripts/calculate_stoploss.py \
  --entry 68500 \
  --trailing 1.5
```

**规则**：
- 价格上涨 1%，止损上移 1%
- 价格下跌，止损不动
- 最大亏损锁定在 1.5%

#### 3. 技术止损

基于支撑/阻力位设置止损：

- **做多**：止损设在支撑位下方 1-2%
- **做空**：止损设在阻力位上方 1-2%

### 止损设置建议

| 时间框架 | 止损幅度 | 止盈幅度 | 盈亏比 |
|----------|----------|----------|--------|
| 1m (剥头皮) | 0.5-1% | 1-2% | 2:1 |
| 5m (日内) | 1.5-2% | 3-4% | 2:1 |
| 15m+ (趋势) | 2-3% | 4-6% | 2:1 |

## 资金管理

### 资金分配

```
总资金：100%
├── 交易仓位：30% (最多)
├── 保证金储备：40%
├── 应急资金：20%
└── 利润提取：10% (定期)
```

### 加仓策略

**金字塔加仓**（推荐）：

```
第一笔：50% 仓位 (试探)
  ↓ 盈利 2%
第二笔：30% 仓位 (确认)
  ↓ 盈利 4%
第三笔：20% 仓位 (加仓)
```

**禁止**：
- ❌ 亏损后加仓（摊平成本）
- ❌ 一次性满仓
- ❌ 借钱交易

### 利润提取

**建议规则**：
- 每周提取利润的 20-30%
- 每月提取利润的 50%
- **落袋为安**

## 风控规则清单

### 开仓前检查

- [ ] 止损价格已计算
- [ ] 仓位大小符合 2% 规则
- [ ] 杠杆不超过 20 倍（新手）
- [ ] 总仓位不超过 30%
- [ ] 盈亏比至少 2:1

### 持仓中监控

- [ ] 每 30 分钟检查一次仓位
- [ ] 价格接近止损时准备手动平仓
- [ ] 盈利 2% 后设置保本止损
- [ ] 关注资金费率（避免高成本）

### 平仓规则

**必须平仓的情况**：
- 触及止损 → **立即平仓**
- 策略信号反转 → **立即平仓**
- 单笔亏损达 2% → **立即平仓**
- 当日亏损达 5% → **停止交易**

**建议平仓的情况**：
- 触及止盈 → 平仓 50%，剩余追踪止损
- 持仓超过 2 小时无进展 → 考虑平仓
- 市场突发重大新闻 → 减仓或平仓

## 心理控制

### 交易纪律

1. **严格执行止损** - 不要抱有幻想
2. **不追涨杀跌** - 等待明确信号
3. **不报复性交易** - 亏损后冷静
4. **不贪婪** - 达到目标就止盈

### 情绪管理

**亏损后**：
- 休息 30 分钟
- 复盘错误
- 不要立即开新仓

**盈利后**：
- 不要过度自信
- 继续按策略执行
- 定期提取利润

### 交易日志

记录每笔交易：

```markdown
- 入场：68,500
- 出场：69,200

- 策略：combo
- 备注：信号明确，执行良好
```
