## 📊 交易策略框架

### 入场信号
- **趋势跟踪**: MA 交叉、MACD 交叉
- **均值回归**: RSI 超买超卖、布林带突破
- **动量策略**: 价格突破、成交量放大

### 出场信号
- **反向信号**: 策略信号反转
- **止损**: 固定百分比 (2-3%) 或 ATR 止损
- **止盈**: 固定百分比 (5-10%) 或移动止盈
- **时间止损**: 超过预期时间未盈利

### 风险管理
- 单笔交易 ≤ 总资金 5%
- 总杠杆 ≤ 5 倍
- 同时持仓 ≤ 3 个品种
- 每日最大亏损 ≤ 10%

## ⚠️ 风险评估

### 技术风险
| 风险 | 缓解措施 |
|------|----------|
| API 限流 | 本地缓存数据，实现重试机制 |
| 网络中断 | 心跳检测，断线重连 |
| 代码 Bug | 充分回测，先用模拟盘 |

### 市场风险
| 风险 | 缓解措施 |
|------|----------|
| 极端行情 | 严格止损，控制杠杆 |
| 流动性不足 | 选择主流币种 |
| 交易所风险 | 选择大交易所，分散资金 |

### 中国大陆用户特别注意
1. 准备稳定的网络代理服务
2. 优先选择 OKX 等对大陆友好的交易所
3. 仅限学习和模拟交易，遵守法律法规
4. API 密钥禁止提现权限，启用 2FA
