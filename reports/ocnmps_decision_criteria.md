# OCNMPS 灰度决策标准

## 首周升到 50% 的条件（全部满足）

| 条件 | 阈值 |
|------|------|
| 平均评分 | ≥ 4.3/5 |
| 回退率 | ≤ 5% |
| 刹车条件 | 未触发 |
| 意图覆盖 | ≥ 3/4 类型有样本 |
| 无高频误路由 | 单意图误路由 ≤ 2 条 |

---

## 维持 30% 的条件

| 条件 | 说明 |
|------|------|
| 平均评分 | 4.0 ~ 4.3 之间 |
| 回退率 | 5% ~ 10% 之间 |
| 有轻微警告 | 但未触发刹车 |
| 某意图样本不足 | 需继续观察 |

---

## 触发刹车回退的条件（任一）

| 条件 | 阈值 |
|------|------|
| 平均评分连续低 | < 4.0 连续 ≥ 2 天 |
| 回退率 | > 10% |
| 延迟严重 | > 500ms |
| 刹车条件被触发 | 见 `BRAKE_CONDITIONS` |

---

## 决策流程

```
周报生成
    ↓
检查刹车条件
    ↓
┌─────────────────────────────────────┐
│  触发刹车？                          │
│  ├─ 是 → 回退到 0%，排查问题          │
│  └─ 否 → 检查升配条件                │
│           ├─ 全部满足 → 升到 50%      │
│           └─ 部分满足 → 维持 30%      │
└─────────────────────────────────────┘
```
