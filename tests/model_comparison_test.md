### 模型 A: bailian/qwen3.5-plus (当前默认)

**题目 1 结果**:
```python
def max_profit_two_transactions(prices: List[int]) -> int:
    if len(prices) < 2:
        return 0
    
    # 第一次交易的最小买入价和最大利润
    buy1 = float('inf')
    profit1 = 0
    # 第二次交易的最小买入价和最大利润
    buy2 = float('inf')
    profit2 = 0
    
    for price in prices:
        # 更新第一次交易
        buy1 = min(buy1, price)
        profit1 = max(profit1, price - buy1)
        
        # 更新第二次交易（用第一次的利润抵消部分成本）
        buy2 = min(buy2, price - profit1)
        profit2 = max(profit2, price - buy2)
    
    return profit2
```
**时间复杂度**: O(n) | **空间复杂度**: O(1)
