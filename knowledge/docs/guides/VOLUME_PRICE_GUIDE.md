# 📊 量价关系完全指南

> 成交量是真相，价格可能是谎言

---

## 💡 核心理念

**量价关系是技术分析的核心**

- **成交量** = 市场参与度（真相）
- **价格** = 市场结果（可能 manipulated）
- **量价配合** = 趋势可靠性

**权重分配**:
- 量价关系：**60%**
- 其他指标（EMA/RSI/MACD）：**40%**

---

```

- 价格上涨 > 0.3%
```

### 7. 顶背离 🔴 反转信号

```
特征:
- 价格创新高
- 成交量低于前高
- 上涨动能衰竭

信号：-3 分
操作：平多仓，准备做空
置信度：75%

示例:
前高：$100, 成交量 100 万
现高：$102, 成交量 60 万
→ 顶背离，即将反转
```

**为什么可靠**:
- 价格上涨但没有量支撑
- 主力在悄悄出货
- 反转概率大

---

- 价格下跌但有人接盘
- 主力在悄悄建仓
- 反弹概率大

## ⚠️ 注意事项

### 1. 成交量基准

**使用 20 周期均线**:
```python
volume_ma_20 = df['volume'].rolling(window=20).mean()
volume_ratio = current_volume / volume_ma_20
```

**为什么 20 周期**:
- 太短（5-10）：噪音大
- 太长（50-100）：反应慢
- 20 周期：平衡

### 2. 时间框架

**推荐**: 1 分钟 - 5 分钟

| 时间框架 | 适用 | 量价信号可靠性 |
|----------|------|----------------|
| 1m | 剥头皮 | 70% |
| 5m | 日内波段 | 80% |
| 15m | 短线 | 85% |
| 1h | 中线 | 90% |

### 3. 标的选择

**高流动性标的量价更可靠**:

| 标的 | 流动性 | 量价可靠性 |
|------|--------|------------|
| BTC/USDT | ⭐⭐⭐⭐⭐ | 95% |
| ETH/USDT | ⭐⭐⭐⭐⭐ | 90% |
| SOL/USDT | ⭐⭐⭐⭐ | 85% |
| 小币种 | ⭐⭐ | 60% (易操纵) |

### 4. 假放量识别

**假放量特征**:
- 突然 1 根 K 线巨量，前后正常
- 价格没有明显波动
- 可能是交易所刷量

**真放量特征**:
- 连续 2-3 根 K 线放量
- 价格配合波动
- 多个交易所同步

## 💡 关键要点

### ✅ 必记

1. **放量上涨** → 做多（最可靠）
2. **缩量上涨** → 观望（可能见顶）
3. **放量下跌** → 做空（最可靠）

3. **背离时逆势** - 与信号对着干
4. **假放量追单** - 被刷量欺骗
