## 交易策略

### 1. 限价单挂单策略 ⭐⭐⭐⭐⭐

**核心理念**: 
- 日常使用限价单（节省 60% 手续费）
- 紧急情况使用市价单（保证执行）

**挂单价格**:
```
做多挂单 = 当前价 × (1 - 0.1%)
做空挂单 = 当前价 × (1 + 0.1%)
```

**优势**:
- 💰 Maker 费率 0.02% (vs Taker 0.05%)
- 💰 避免滑点
- 💰 控制入场节奏

---

### 2. 分批建仓策略 ⭐⭐⭐⭐⭐

**分批逻辑**:
```
总资金：$1000

第 1 批：$500 (50%) @ -0.1%
第 2 批：$300 (30%) @ -0.3%
第 3 批：$200 (20%) @ -0.5%

均价：-0.26% (优于市价)
```

**优势**:
- ✅ 降低单次入场风险
- ✅ 获得更好均价
- ✅ 避免一次性高位入场

---

### 3. 强制止损策略 ⭐⭐⭐⭐⭐

**止损配置**:
| 参数 | 值 | 说明 |
|------|-----|------|
| **止损类型** | 硬止损 | 市价单保证执行 |
| **止损比例** | 0.5% | 亏损 0.5% 平仓 |
| **警报级别** | 3 级 | 0.3%/0.4%/0.5% |

**止损流程**:
```
开仓 → 设置止损单
  ↓
监控价格 (10 秒/次)
  ↓
触及止损价
  ↓
市价单立即平仓
  ↓
记录止损原因
```

---

### 4. 智能撤单策略 ⭐⭐⭐⭐

**撤单条件**:
| 条件 | 阈值 | 动作 |
|------|------|------|
| **超时** | >3 分钟 | 撤单重挂 |
| **信号失效** | 指标反转 | 立即撤单 |
| **市场变化** | >1% | 撤单重评 |
| **紧急** | 黑天鹅 | 撤单 + 平仓 |

---

### 5. 信号生成策略

**信号来源**:
- 技术指标 (EMA/RSI/MACD/KDJ)
- 量价关系 (放量/缩量)
- 市场情绪 (恐惧贪婪指数)
- 宏观事件 (财经日历)

**信号融合**:
```
综合评分 = 技术面×40% + 预测面×30% + 情绪×20% + 宏观×10%

STRONG_BUY: 评分>0.75, 置信度>0.7
BUY: 评分>0.6, 置信度>0.5
HOLD: -0.6<评分<0.6
SELL: 评分<-0.6, 置信度>0.5
STRONG_SELL: 评分<-0.75, 置信度>0.7
```

## 风险管理系统

### 1. 四层风控保护

```
┌─────────────────────────────────┐
│   黑天鹅防护 (最外层)            │
│   5 分钟跌>5% → 立即清仓         │
└─────────────────────────────────┘
              ↓
┌─────────────────────────────────┐
│   硬止损 (每单保护)              │
│   0.5% 亏损 → 市价平仓           │
└─────────────────────────────────┘
              ↓
┌─────────────────────────────────┐
│   网络保护 (60 秒无响应)          │
│   紧急平仓 → 保护本金            │
└─────────────────────────────────┘
              ↓
┌─────────────────────────────────┐
│   保证金监控 (>90% 警报)          │
│   自动减仓 → 避免爆仓            │
└─────────────────────────────────┘
```

---

### 2. 网络波动保护

**重试机制**:
```
API 请求失败
  ↓
重试 (最多 3 次，间隔 5 秒)
  ↓
重试失败
  ↓
检查持仓状态
  ↓
亏损>0.4%？
  ↓
是 → 紧急市价平仓
否 → 等待网络恢复
```

---

### 3. 黑天鹅防护

**检测维度**:
- 闪崩检测：5 分钟跌幅>5%
- 波动率暴增：ATR>3 倍均值
- 成交量异常：>5 倍均值
- 资金费率极端：>0.1%

**警报级别**:
| 级别 | 条件 | 动作 |
|------|------|------|
| 🟢 GREEN | 无异常 | 正常交易 |
| 🟡 YELLOW | 成交量 3-5x | 减仓 20-30% |
| 🟠 ORANGE | 5 分钟跌 3-5% | 减仓 50% |
| 🔴 RED | 5 分钟跌>5% | 立即清仓 |

```json
{
  "symbol": "BTC/USDT:USDT",
  "side": "buy",
  "type": "limit",
  "amount": 0.01,
  "price": 70000,
  "stop_loss": {
    "enabled": true,

    "order_type": "market"

    "enabled": true,
    "price": 71050,
```

**关键配置**:
- `stop_loss.enabled`: **必须为 true**
- `stop_loss.type`: **hard** (硬止损)
- `stop_loss.order_type`: **market** (市价单)

**验证项目**:
- ✅ 止损是否设置
