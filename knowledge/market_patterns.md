## 成交量形态

### 量价关系
| 关系 | 特征 | 信号 |
|------|------|------|
| 放量上涨 | 价涨量增 | 健康上涨 |
| 缩量上涨 | 价涨量减 | 可能见顶 |
| 放量下跌 | 价跌量增 | 恐慌抛售 |
| 缩量下跌 | 价跌量减 | 可能见底 |

### 成交量异常
| 异常 | 特征 | 信号 |
|------|------|------|
| 天量 | >5 倍均量 | 变盘前兆 |
| 地量 | <0.5 倍均量 | 整理末期 |
| 量堆 | 连续放量 | 主力进场 |

## 快速判断

### 趋势判断
```
上涨趋势：高点更高，低点更高
下跌趋势：高点更低，低点更低
震荡趋势：高点和低点水平
```

### 强弱判断
```
强势：站稳 EMA9，沿 EMA9 上涨
中性：在 EMA20 附近震荡
弱势：跌破 EMA50，沿 EMA50 下跌
```

### 买卖点
```
买点：
1. 突破关键阻力 + 放量
2. 回踩支撑 + 缩量
3. 指标背离 + 形态确认

卖点：
1. 跌破关键支撑 + 放量
2. 反抽阻力 + 缩量
3. 指标背离 + 形态确认
```
