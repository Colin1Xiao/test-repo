## 📊 交易逻辑

### 买入信号
```

置信度 = min(涨幅/3, 100%)

```

### 卖出信号
```
条件：价格下跌 > 1%
置信度 = min(跌幅/3, 100%)
执行：置信度 > 70% 时卖出
```
