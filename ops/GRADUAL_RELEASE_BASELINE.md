## 五、灰度期禁止改动范围

### 🔴 禁止 (除非紧急修复)

| 模块 | 禁止内容 | 原因 |
|------|---------|------|
| **Runtime 主链** | buildSkill / ExecutionContext / ToolRegistry | 核心底座，改动风险极高 |
| **QueryGuard 状态机** | Turn 状态机 (idle/dispatching/running) | 行为稳定性基础 |
| **PermissionEngine 主判定** | 规则匹配逻辑 / 决策合并算法 | 安全核心，不可动 |
| **TaskStore / ApprovalStore** | 数据结构 / 状态流转 | 任务一致性保障 |
| **HookBus 核心** | 事件发布/订阅机制 | 解耦基础 |
| **MemDir 读写接口** | 文件型记忆 API | 长期一致性保障 |

## 六、回滚策略

### 回滚触发条件

| 级别 | 条件 | 动作 |
|------|------|------|
| **Critical** | 数据损坏 / 安全漏洞 / 系统崩溃 | 立即回滚 |
| **Warning** | 成功率 < 90% / 审批卡死 > 5 例 | 暂停灰度，评估后决定 |
| **Info** | 非关键 bug / 用户体验问题 | 记录，下个版本修复 |

### 回滚目标版本

| 场景 | 回滚目标 | 备注 |
|------|---------|------|
| Runtime 问题 | `2026.3.12` (稳定版) | 升级前版本 |
| 插件问题 | 禁用对应插件 | 不影响核心 |
| 配置问题 | 恢复默认配置 | 保留用户数据 |
