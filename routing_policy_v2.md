# OpenClaw 路由策略 V2

**版本：** v2.0  
**生效时间：** 2026-03-13  
**状态：** 正式运行

---

## 目标

减少不必要的混合编排和高成本路由，优化任务分配。

---

## 任务白名单 / 黑名单

### 强制单模型（禁止混合链路）

| 条件 | 路由目标 | 原因 |
|------|----------|------|
| 输入长度 < 50 字 | FAST | 简单问题不需要混合 |
| 纯事实问答 | FAST | 不需要多步推理 |
| 单一代码函数 | CODE | 直接生成更高效 |
| 纯翻译任务 | CN | 单模型足够 |

### 强制预处理（先摘要再编排）

| 条件 | 处理方式 | 原因 |
|------|----------|------|
| 输入长度 > 5000 字 | LONG 先摘要 | 减少后续步骤负担 |
| 多文档输入 | LONG 先整合 | 统一格式后再处理 |
| 复杂日志 | LONG 先提取关键信息 | 去噪 |

### 强制 MAIN 兜底

| 条件 | 处理方式 | 原因 |
|------|----------|------|
| 高不确定性任务 | MAIN 直接处理 | 避免链路累积误差 |
| 用户明确要求简单回答 | MAIN | 避免过度处理 |
| 任务类型无法识别 | MAIN | 安全兜底 |

## 混合链路触发条件

### B1: 调试 + 修复

**触发条件：**
- [ ] 包含报错信息
- [ ] 包含代码片段
- [ ] 明确要求"分析并修复"

**禁止触发：**
- 仅要求"解释报错"
- 仅要求"分析原因"
- 无代码片段

### B2: 长文 + 推理

**触发条件：**
- [ ] 输入长度 > 1000 字
- [ ] 包含文档/会议纪要
- [ ] 需要架构/方案决策

**禁止触发：**
- 简单问答
- 纯技术问题
- 输入 < 500 字

### B3: 推理 + 中文润色

**触发条件：**
- [ ] 需要方案比较
- [ ] 需要正式汇报
- [ ] 输入为结构化分析

**禁止触发：**
- 简单改写
- 非正式场景
- 无推理需求

## 重试策略细化

### 错误类型到重试策略映射

| 错误类型 | 可重试 | 重试次数 | 退避策略 | fallback |
|----------|--------|----------|----------|----------|
| empty_response | ✅ | 1 | 立即 | 允许 |
| provider_slow | ✅ | 2 | 2s 退避 | 允许 |
| provider_error (5xx) | ✅ | 2 | 5s 退避 | 允许 |
| provider_error (4xx) | ❌ | 0 | - | 必须 |
| timeout | ❌ | 0 | - | 必须 |
| subagent_failed | ✅ | 1 | 立即 | 允许 |
| unknown | ❌ | 0 | - | 允许 |

### 混合链路重试特殊规则

- 单步重试次数 ≤ 2 次
- 链路总重试次数 ≤ 3 次
- 重试失败后必须 fallback，不能无限重试
- MAIN 汇总步骤不重试，直接 fallback

## 路由决策流程 V2

```
接收任务
    ↓
判断输入长度
    ↓
< 50 字? → FAST (强制单模型)
    ↓ 否
> 5000 字? → LONG 预处理
    ↓ 否
判断任务类型
    ↓
纯代码? → CODE/CODE-PLUS
纯中文? → CN
纯推理? → REASON
    ↓
混合任务?
    ↓ 是
检查触发条件
    ↓ 满足
进入混合链路
    ↓ 不满足
MAIN 兜底
```

## 成本优化策略

### 高成本低收益任务识别

| 特征 | 处理方式 |
|------|----------|
| 简单任务进入复杂链路 | 降级到单模型 |
| 长任务未压缩 | 强制预处理 |
| 频繁 fallback | 调整路由策略 |
| 高耗时低质量 | 标记优化 |

### 模型成本分级

| 模型 | 成本等级 | 使用策略 |
|------|----------|----------|
| FAST | 低 | 优先使用 |
| CODE | 中 | 代码任务专用 |
| LONG | 中 | 长文本专用 |
| REASON | 高 | 复杂推理专用 |
| GROK-CODE | 高 | Debug 专用 |
| CODE-PLUS | 高 | 重构专用 |
| CN | 中 | 润色专用 |
| MAIN | 中 | 兜底使用 |
