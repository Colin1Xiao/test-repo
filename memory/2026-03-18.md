### 执行验证阶段纪律

1. **第一笔不是结论，是样本模板**
2. **验证的是执行质量，不是盈利能力**
3. **系统必须通过市场，不是市场配合系统**
4. **不要试图赢市场，先证明系统不会输给执行**

- 价格跳跃保护 (>0.05%放弃)
- 延迟分析 (信号→下单延迟)
- 市场基准 (bid/ask/mid价格记录)
- 滑点计算基于盘口价
