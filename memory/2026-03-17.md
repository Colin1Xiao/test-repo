## 技术总结

### launchd配置要点
- `RunAtLoad`: true (启动时立即运行)
- `KeepAlive`: true (崩溃后自动重启)
- `StandardOutPath/StandardErrorPath`: 日志追加模式
- `ThrottleInterval`: 5秒 (重启间隔)

### 评分引擎架构
- 独立模块可单测
- 输入: OHLCV数据 + 市场状态
- 输出: 评分明细 + 是否达标
- 可解释性强

### 影子模式原则
- 不改执行层
- 不改风控参数
- 不接真实下单
- 仅记录候选信号
