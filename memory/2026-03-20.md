### 🎯 Decision Hub V2 核心设计

**原则**："没有任何路径可以绕过 Decision Hub"

**三层防绕过**:
```
Layer 1: Execution Gate → 只接受 ExecutionRequest
Layer 2: ExecutionRequest → 必须包含授权 Decision
Layer 3: Decision → 必须来自 DecisionHub.evaluate()
```

**强制规则**:
```python
# ❌ 绕过 Hub → UnauthorizedExecutionError
executor.execute_signal(...)

# ✅ 唯一授权路径
decision = hub.evaluate(signal)
request = ExecutionRequest(decision)
gate.execute(request)
```
