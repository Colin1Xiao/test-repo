## 一、项目概述

### 1.1 项目背景

小龙自动交易系统是一个基于多因子信号融合、支持挂单进场/平仓、具备完善风控体系的加密货币自动交易系统。系统经过V3版本22项检查验证和V3.1执行层改造，目前已进入模拟盘连续验证阶段。

### 1.2 核心特性

- **多因子信号融合**: 技术面40% + ML预测30% + 情绪25% + 宏观5%
- **挂单执行模式**: 默认限价挂单进场/平仓，紧急风控例外
- **分层风控体系**: 日$500/周$2000/月$8000/回撤10%
- **执行状态机**: 10种状态完整流转，支持部分成交
- **多交易所适配**: OKX主用，Binance备用（当前单OKX模式）
- **7×24监控**: 自动重启，Telegram告警

## 九、风险提示

### 9.1 单交易所风险

- 系统完全依赖OKX单一交易所
- OKX维护或故障时系统不可用
- 已配置监控和告警，异常时通知

### 9.2 高杠杆风险

- 20x杠杆会放大收益和亏损
- 5%反向波动即爆仓
- 严格止损，控制单笔风险

### 9.3 策略风险

- 历史回测不代表未来收益
- 市场环境变化可能导致策略失效
- 持续监控，及时调整
