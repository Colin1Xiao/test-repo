# 华为 HN8156XR-10 端口映射配置指南

## 登录路由器

1. 打开浏览器访问: http://192.168.1.1 或 http://192.168.0.1
2. 用户名: `admin` 或 `user`
3. 密码: 路由器背面标签上的默认密码，或你自己设置的密码

> 如果无法登录，尝试: http://192.168.100.1 (华为光猫常用)

---

## 配置步骤

### 方法 1: Web 界面配置（推荐）

**步骤 1: 进入高级设置**
```
登录后 -> 点击"高级"或"高级设置" -> 可能需要再次输入密码
```

**步骤 2: 找到端口映射/虚拟服务器**
```
路径可能为:
- 高级 -> 网络 -> NAT -> 端口映射
- 高级 -> 安全 -> 端口转发
- 高级 -> 应用 -> 虚拟服务器
```

**步骤 3: 添加端口映射规则**

点击"添加"或"新建"，填写:

```
名称: Control-Tower
协议: TCP
外部端口: 8767
内部端口: 8767
内部 IP: 192.168.1.65
状态: 启用
```

**步骤 4: 保存并应用**
```
点击"保存"或"应用"
等待路由器重启（如果需要）
```

---

### 方法 2: Telnet/SSH 配置（高级）

如果 Web 界面找不到，可以尝试命令行:

```bash
# 1. 启用 Telnet（如果支持）
# 在 Web 界面找到 Telnet 开关并启用

# 2. 连接路由器
telnet 192.168.1.1

# 3. 登录后执行（命令可能不同）
system-view
nat server protocol tcp global current-port 8767 inside 192.168.1.65 8767
save
```

## 常见问题

### Q: 找不到端口映射选项
```
华为光猫可能有以下情况:
1. 运营商定制版 - 功能被隐藏
   解决: 尝试访问 http://192.168.100.1 或 http://192.168.1.1:8080

2. 需要超级管理员权限
   尝试用户名: telecomadmin / CMCCadmin
   密码: nE7jA%5m 或查看光猫背面

3. 功能名称不同
   查找: NAT, 虚拟服务器, 端口转发, DMZ, UPnP
```

### Q: 端口映射后仍无法访问
```
1. 检查 Mac 防火墙
   系统设置 -> 网络 -> 防火墙 -> 关闭

2. 检查运营商是否封锁端口
   尝试更换端口: 8080, 8888, 9000

3. 检查是否获取到公网 IP
   对比路由器 WAN IP 和 api.ipify.org 返回的 IP
   如果不一致，可能是运营商 NAT
```

### Q: 路由器重启后配置丢失
```
保存配置后，有些路由器需要:
1. 点击"保存到配置文件"
2. 或"系统工具" -> "保存配置"
```

## 备用方案

如果华为光猫无法配置端口映射，可以:

### 方案 A: 使用 UPnP（自动端口映射）
```bash
# 安装 miniupnpc
brew install miniupnpc

# 自动映射端口
upnpc -a 192.168.1.65 8767 8767 tcp
```

### 方案 B: 改用 frp/ngrok
```bash
# 回到之前的方案
./start-frp.sh
# 或
./start-public.sh ngrok
```

### 方案 C: DMZ 主机（不推荐，安全性低）
```
在路由器设置 DMZ 主机为 192.168.1.65
这样所有端口都会映射到本机
```
