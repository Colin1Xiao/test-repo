## 🔧 关键修复

### V5.4 关键修复列表

1. **StateStore 持仓恢复**
   - 问题：重启后 `_current_position` 丢失
   - 修复：从文件读取 `last_event` 恢复状态

2. **条件单验证 API**
   - 问题：无法查询条件单状态
   - 修复：使用 `ordType: conditional` 参数

3. **从 info 读取参数**
   - 问题：ccxt 返回的条件单参数位置不明
   - 修复：从 `info` 字典读取参数

4. **stop_loss_result 传递**
   - 问题：止损结果未传递到门禁
   - 修复：合并到 `gate_snapshot` 中
