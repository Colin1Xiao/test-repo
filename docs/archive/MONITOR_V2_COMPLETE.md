## 🚀 使用方法

### 启动监控

```bash
cd /Users/colin/.openclaw/workspace

# 基础监控
python3 auto_monitor_v2.py

# 指定币种
python3 auto_monitor_v2.py --symbols BTC/USDT,ETH/USDT

# 指定间隔
python3 auto_monitor_v2.py --interval 30

# 禁用声音
python3 auto_monitor_v2.py --no-sound
```

### 后台运行

```bash
# 后台启动
nohup python3 auto_monitor_v2.py > monitor_v2.log 2>&1 &

# 查看进程
ps aux | grep auto_monitor_v2

# 查看日志
tail -f monitor_v2.log
tail -f monitor_alerts.log

# 停止监控
pkill -f auto_monitor_v2
```
