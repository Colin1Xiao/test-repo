## 😴 睡眠与休息管理

### 单人交易员睡眠方案

**推荐：02:00-08:00 睡眠（6 小时）**

```
理由:
- 02:00-05:00 美盘尾盘，波动大但风险高
- 05:00-08:00 凌晨低波动，机器主导
- 错过 6 小时约 2-4 次交易机会
- 但保证充足睡眠，避免疲劳交易

睡前准备:
1. 平掉所有仓位（不要持仓过夜）
2. 设置价格警报（±2% 波动）
3. 关闭交易软件，避免 temptation
4. 保证 6-7 小时睡眠
```

**备选：00:00-06:00 睡眠**

```
优点:
- 抓住美盘高峰（23:00-00:00）
- 早起抓亚洲早盘（06:00-08:00）

缺点:
- 睡眠时间短（6 小时）
- 可能疲劳
```

## ⚠️ 24 小时交易风险

### 1. 疲劳交易

**症状**:
- 判断力下降
- 反应变慢
- 情绪波动
- 频繁犯错

**预防**:
```
✓ 保证 6-7 小时睡眠
✓ 每 2 小时休息 15 分钟
✓ 每小时起身活动
✓ 多喝水，少吃甜食
✓ 限制咖啡因（每日<400mg）
```

### 2. 过度交易

**症状**:
- 为了交易而交易
- 低质量信号也入场
- 交易次数>15 次/天
- 手续费吃掉利润

**预防**:
```
✓ 设定每日最大交易次数（10-12 次）
✓ 只交易 A 级/S 级信号
✓ 记录每笔交易原因
✓ 定期复盘，剔除低质量交易
```

### 3. 错过休息

**症状**:
- 连续交易>18 小时
- 不吃饭/不上厕所
- 精神亢奋但判断力下降
- 家人/朋友抱怨

**预防**:
```
✓ 设定闹钟提醒休息
✓ 固定吃饭时间
✓ 每天运动 30 分钟
✓ 每周休息 1 天（或不交易）
```

### 4. 持仓过夜风险

**风险**:
- 睡觉时大幅波动
- 爆仓无人管
- 止损可能跳空

**预防**:
```
✓ 睡前平掉所有仓位
✓ 或设置硬止损单（交易所级别）
✓ 或降低仓位至 5-10%
✓ 或使用对冲（做空等量）
```
