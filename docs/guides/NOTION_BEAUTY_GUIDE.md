## 🎨 设计原则

### 1. 专业配色方案

| 颜色 | 用途 | 情感 |
|------|------|------|
| **绿色** | 盈利/上涨 | 积极、安全 |
| **红色** | 亏损/风险 | 警示、紧急 |
| **蓝色** | 信息/状态 | 专业、信任 |
| **紫色** | 持仓/资产 | 高端、稳定 |
| **橙色** | 信号/机会 | 活力、机会 |
| **灰色** | 中性/等待 | 平静、中性 |

---

### 2. 关键指标优先

**首屏展示**（最重要的数据）:
```

│ 💰 总资金：$500                     │
│ 📈 总盈亏：$0 (0.00%)               │
```

**设计理由**:
- ✅ 一眼看到核心数据

- ✅ emoji 图标增强识别

---

### 3. 分区清晰

**7 个核心区域**:

1. **📊 关键指标** - 核心数据卡片
2. **🔴 实时监控** - 系统状态

6. **⚡ 快速操作** - 常用功能

## 🎨 手动美化技巧

### 1. 添加彩色背景

在 Notion 中：
1. 点击内容块左侧的 `⋮⋮`
2. 选择 "Color"
3. 选择背景色：
   - 绿色背景 → 盈利数据
   - 红色背景 → 风险警告
   - 蓝色背景 → 信息提示
   - 灰色背景 → 中性数据

---

### 2. 使用 Callout 块

Callout 是最美的展示方式：
1. 输入 `/callout`
2. 选择 emoji 图标
3. 输入内容
4. 设置背景色

**示例**:
```
💡 Callout 示例
┌────────────────────────────┐
│ 💡 这是一条重要信息        │
│ [蓝色背景]                 │
└────────────────────────────┘
```

---

### 3. 添加分隔线

让页面更清晰：
1. 输入 `---` 或 `***`
2. 自动变成分隔线
3. 分隔不同区域

---

### 4. 使用 Toggle 列表

折叠不常用的内容：
1. 输入 `> ` (大于号 + 空格)
2. 创建可折叠列表
3. 点击箭头展开/折叠

**示例**:
```
▶️ 历史交易记录 (点击展开)
  [交易 1]
  [交易 2]
  [交易 3]
```

---

### 5. 添加进度条

使用 Notion 公式：
1. 添加 Number 字段
2. 设置为 "Bar" 或 "Ring" 显示
3. 自动显示进度条

**示例**:
```
目标进度：[████████░░] 80%
```

## 🎯 最佳实践

### ✅ 推荐做法

1. **保持简洁** - 不要过度装饰
2. **颜色一致** - 同类型数据用同色
3. **层次清晰** - 重要数据放上面
4. **定期更新** - 保持数据新鲜
5. **备份配置** - 保存模板

---

### ❌ 避免做法

1. **颜色过多** - 超过 5 种颜色
2. **信息过载** - 首屏太多数据
3. **布局混乱** - 没有清晰分区
4. **数据过时** - 长时间不更新
5. **字体杂乱** - 使用过多字体

---

## 📱 移动端优化

### 手机查看优化
