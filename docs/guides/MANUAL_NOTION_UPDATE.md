### 步骤 3: 编辑卡片内容

#### 编辑本金卡片

1. 点击绿色背景的本金卡片
2. 修改内容为：
   ```
   💰 初始本金：$500 | 当前本金：$[你的真实金额]
   ```
3. 按 `Enter` 保存

#### 编辑盈亏卡片

1. 点击灰色背景的盈亏卡片
2. 修改内容为：
   ```
   📈 总盈亏：$[盈亏金额] ([盈亏比例]%)
   ```
3. 按 `Enter` 保存

**计算方式**:
```
盈亏金额 = 当前本金 - 初始本金
盈亏比例 = (盈亏金额 / 初始本金) × 100%
```

**示例**:
```
当前本金：$650
初始本金：$500
盈亏金额：$150
盈亏比例：30%

卡片内容：📈 总盈亏：$150 (+30.00%)
```

---

### 步骤 4: 更新目标进度

1. 找到紫色背景的目标进度卡片
2. 点击编辑
3. 修改进度条：

**进度条计算**:
```
目标：$500 → $100,000
已实现：$[当前本金 - 500]
进度比例：[(当前本金 - 500) / 99500] × 100%
```
