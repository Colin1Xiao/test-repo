## 💡 核心优化

### 1. 精确买入时机

**不再模糊入场，必须同时满足**:

```
✅ 趋势确认（EMA 排列）
✅ 动量确认（RSI/MACD）
✅ 成交量确认（>1.2x 均值）
✅ 置信度 > 60%（9 项指标评分≥9 分）
```

**评分系统**（满分 15 分）:
| 指标 | 权重 | 得分条件 |
|------|------|----------|
| EMA 排列 | 4 分 | 多头/空头排列 |
| RSI 超卖/超买 | 3 分 | <30 或 >70 |
| KDJ 金叉/死叉 | 2 分 | K 穿越 D |
| MACD 金叉/死叉 | 2 分 | MACD 穿越信号线 |
| 成交量放大 | 2 分 | >1.5x 均值 |
| 动量确认 | 2 分 | 3 日/5 日动量 |

**≥9 分**: 入场信号  
**≥12 分**: 强烈信号（50x 杠杆）

### 2. 精确卖出时机

**6 种出场信号**:

| 类型 | 条件 | 动作 |
|------|------|------|
| **止盈** | 盈利≥30% (50x 杠杆抓 0.6%) | 平仓 50%，剩余追踪 |
| **回撤** | 从高点回撤>0.5% | 全部平仓 |
| **RSI 极端** | RSI>75(多)/RSI<25(空) | 全部平仓 |
| **MACD 反转** | MACD 反向交叉 | 全部平仓 |
| **硬止损** | 亏损 -25% (0.5% 价格) | 立即平仓 |
| **时间止损** | 30 分钟无盈利 | 考虑平仓 |

## 🎯 买卖时机详解

### 最佳买入时机（做多）

**S 级信号**（置信度≥80%）:
```
✓ EMA5 > EMA9 > EMA20（多头排列）
✓ RSI(7) < 35 或从<30 反弹
✓ KDJ 金叉（K 从下向上穿越 D）
✓ MACD > Signal 且 MACD 刚金叉
✓ 成交量 > 1.5x 均值
✓ 3 日动量 > 0.3%
✓ 价格在 EMA5 上方

出现 5 个以上 → 强烈买入（50x 杠杆）
```

**A 级信号**（置信度 60-80%）:
```
✓ EMA 多头排列
✓ RSI 40-50（中性偏强）
✓ 成交量 > 1.2x
✓ 至少 3 个指标共振

出现 3-4 个 → 买入（30-40x 杠杆）
```

### 最佳买入时机（做空）

**S 级信号**（置信度≥80%）:
```
✓ EMA5 < EMA9 < EMA20（空头排列）
✓ RSI(7) > 65 或从>70 回调
✓ KDJ 死叉（K 从上向下穿越 D）
✓ MACD < Signal 且 MACD 刚死叉
✓ 成交量 > 1.5x 均值
✓ 3 日动量 < -0.3%
✓ 价格在 EMA5 下方

出现 5 个以上 → 强烈做空（50x 杠杆）
```

---

### 最佳卖出时机

**止盈出场**（盈利≥30%）:
```
条件:
- 价格达到止盈目标（+1-1.2%）
- 杠杆后盈利≥30-60%

动作:
1. 平仓 50%，锁定利润
2. 剩余仓位设置追踪止损（0.5%）
3. 如果继续上涨，追踪止损上移
4. 如果回调，触发追踪止损全平
```

**动量衰竭出场**:
```
条件:
- 从高点回撤>0.5%
- 或从低点反弹>0.5%（空单）

动作:
- 立即全部平仓
```

**RSI 极端出场**:
```
条件:
- 做多：RSI(7) > 75
- 做空：RSI(7) < 25

动作:
- 立即全部平仓
```

**MACD 反转出场**:
```
条件:
- 做多：MACD 死叉（从金叉变死叉）
- 做空：MACD 金叉（从死叉变金叉）

动作:
- 立即全部平仓
```

**硬止损出场**:
```
条件:
- 亏损达到 -25%（0.5% 价格波动）

动作:
- 立即平仓，不要犹豫
```

## 💡 关键要点

### ✅ 必做
1. **只交易高置信度**（≥60%）
2. **严格止损**（-25% 立即平仓）
3. **快速止盈**（+30-60% 分批了结）
4. **选择大波动标的**（BTC/ETH/SOL/DOGE）
5. **双向交易**（不逆势，跟随趋势）

### ❌ 禁止
1. **扛单**（亏损不止损）
2. **重仓**（单笔>30% 仓位）
3. **频繁交易**（每日>15 次）
4. **低置信度交易**（<60% 也入场）
5. **听消息交易**（只相信指标）
