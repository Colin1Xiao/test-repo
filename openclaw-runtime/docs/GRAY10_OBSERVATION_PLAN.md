## 六、告警响应流程

### P0 事件响应

```
1. 告警触发 (自动)
   ↓
2. On-call 确认 (5min)
   ↓
3. 初步评估 (10min)
   ↓
4. 决策：回滚 / 修复
   ↓
5. 执行 (30min)
   ↓
6. 事后分析 (24h)
```

