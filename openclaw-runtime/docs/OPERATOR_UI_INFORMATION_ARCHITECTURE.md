### Incidents (事件列表)

**目的**: 查看和管理所有协调层事件

**布局**:
```
┌─────────────────────────────────────────────────────────────┐
│  Incidents                                 [+ New Filter]   │
├─────────────────────────────────────────────────────────────┤
│  Filters: [All ▼] [Severity ▼] [Type ▼] [Status ▼] [🔍]   │
├─────────────────────────────────────────────────────────────┤
│  ┌───────────────────────────────────────────────────────┐ │
│  │ 🔴 P0  | Lease Owner Drift       | 18:30 | Open      │ │
│  │ 🟠 P1  | High Latency (P99>500ms)| 18:25 | Acked     │ │
│  │ 🟡 P2  | Memory Growth Warning   | 18:20 | Open      │ │
│  │ 🟢 P3  | Stale Cleanup Completed | 18:15 | Resolved  │ │
│  │ 🟢 P3  | Snapshot Created        | 18:00 | Resolved  │ │
│  └───────────────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────────────────┤
│  Page: [1] [2] [3] ... [10]                     500 items  │
└─────────────────────────────────────────────────────────────┘
```

**列表字段**:
| 字段 | 说明 |
|------|------|
| Severity | P0/P1/P2/P3 |
| Title | 事件标题 |
| Timestamp | 发生时间 |
| Status | Open/Acked/Resolved |
| Type | Lease/Item/Suppression/Cleanup |
| Instance | 相关实例 |

## 四、响应式设计

### 桌面端 (≥1200px)

- 三栏布局
- 完整图表和详情
- 多窗口并排

### 平板端 (768px - 1199px)

- 两栏布局
- 简化图表
- 可折叠侧边栏

### 移动端 (<768px)

- 单栏布局
- 关键指标优先
- 底部导航栏
