## 七、灰度 10% 计划

### 部署策略

| 参数 | 值 |
|------|-----|
| 灰度比例 | 10% |
| 观察期 | 7 天 |
| 实例数 | 3 (生产集群) |
| 回滚阈值 | 错误率 > 1% / P0 事件 |

### 监控指标

| 指标 | 阈值 | 告警级别 |
|------|------|---------|
| 错误率 | < 0.1% | P0 if > 1% |
| P99 延迟 | < 200ms | P1 if > 500ms |
| 内存增长 | < 10MB/h | P1 if > 50MB/h |
| Stale Cleanup | 正常触发 | P1 if 失效 |

### 回滚流程

```
1. 检测到 P0 事件 / 错误率 > 1%
   ↓
2. 自动触发告警
   ↓
3. On-call 确认 (5min)
   ↓
4. 执行回滚 (10min)
   ↓
5. 事后分析 (24h 内)
```
