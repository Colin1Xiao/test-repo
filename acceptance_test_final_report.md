## 四、问题与优化方案

### 问题 1：长任务超时

**现象：**
- CODE-PLUS、CN 单任务 60s 超时
- B2 MAIN 汇总 60s 超时

**根因：**
- 默认 60s 超时不足以完成大型代码重构和长文本润色
- MAIN 汇总阶段重复展开前两步内容，导致输出过长

**优化方案（已配置）：**

```python
# 超时策略调整
普通任务：30-60s
长文/重构/汇总：90-120s

# MAIN 汇总限长策略
- 只输出：结论 + 关键理由 + 风险 + 下一步
- 避免重复展开前两步全部内容
- 长度控制：简单任务 200-500字，复杂任务 500-800字
```

### 问题 2：混合任务拆分（已修复）

**现象：** 子代理内部调用 sessions_spawn 失败
**修复：** 主代理直接控制整个调用链
