def check_position_gate(symbol: str) -> bool:
    """检查是否有持仓，有则禁止开新仓"""
    position = exchange.fetch_position(symbol)
    if position and abs(float(position.get('contracts', 0))) > 0:
        logger.warning(f"⛔ 已有持仓 {position['contracts']}，禁止重复开仓")
        return False
    return True

## 🟡 P1 修复清单 (马上做)

### 3. 订单级止损 (交易所托管)

**问题**: 当前止损是"逻辑止损"(轮询触发)，不是"订单级止损"

**风险**: 网络卡顿/程序卡住 → 止损失效

**正确实现**:
```python
def execute_with_stop(exchange, symbol, side, amount):
    """开仓同时设置止损"""
    
    # 1. 开仓
    order = exchange.create_market_order(symbol, side, amount)
    entry_price = order.get("price") or exchange.fetch_ticker(symbol)["last"]
    
    # 2. 计算止损价 (-0.5%)
    if side == "buy":
        stop_price = entry_price * 0.995  # -0.5%
        stop_side = "sell"
    else:
        stop_price = entry_price * 1.005  # +0.5%
        stop_side = "buy"
    
    # 3. 设置止损单 (交易所托管)
    exchange.create_order(
        symbol,
        "stop_market",
        stop_side,
        amount,
        None,
        {
            "stopPrice": stop_price,
            "reduceOnly": True  # ⚠️ 关键: 防止反向开仓
        }
    )
    
    return {
        "entry_price": entry_price,
        "stop_price": stop_price,
        "stop_order_id": stop_order["id"]
    }
```

## ⚠️ 三个关键坑

### 坑 1: 不要只用代码止损

```python
# ❌ 危险: 代码止损
if price < stop:
    close_position()

# ✅ 安全: 交易所托管止损
create_stop_loss_order()
```

### 坑 2: 必须 reduceOnly

```python
# ❌ 可能变成反向开仓
create_order(..., side="sell")

# ✅ 只平仓，不开新仓
create_order(..., side="sell", params={"reduceOnly": True})
```

### 坑 3: 止损必须绑定仓位大小

```python
# ❌ 如果仓位是 0.26，止损只挂 0.13 → 只止一半
stop_amount = 0.13  # 硬编码

# ✅ 止损数量 = 持仓数量
stop_amount = current_position_size
```
