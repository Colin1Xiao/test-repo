# 降级生产模式配置

## 系统定义

```
MODE: DEGRADED_PRODUCTION
PHASE: 2.0 (降级版)
DATA_SOURCE: REST_CACHE (2秒更新)
MARKET_SCOPE: RANGE_ONLY
```

## 交易边界

### ✅ 允许
- RANGE 行情（震荡）
- 结构确认后入场
- 低波动时段

### ❌ 禁止
- BREAKOUT 行情
- NEWS 事件
- 高动量追单
- 秒级机会

## 保护机制

| 保护 | 阈值 | 动作 |
|------|------|------|
| 延迟熔断 | > 1.5s | 拒绝执行 |
| 数据新鲜度 | > 1.0s | 切换 REST |
| 波动率过滤 | > 0.1% | 拒绝执行 |
| 盘口价差 | > 0.05% | 拒绝执行 |
| 滑点预检 | > 0.5% | 拒绝执行 |

## 性能标准（降级版）

| 指标 | 目标 |
|------|------|
| P50 | < 1100ms |
| P90 | < 1400ms |
| Max | < 2000ms |
| CV | < 0.15 |

## 能力评估

| 维度 | 状态 |
|------|------|
| 执行系统 | 🟢 生产级 |
| 风控系统 | 🟢 生产级 |
| 延迟稳定性 | 🟢 稳定 |
| 数据源 | 🟡 降级 |
| 高频能力 | 🔴 不可用 |

## 验证目标

收集 50-100 笔真实交易，分析：

1. **盈亏分布**
   - 平均 PnL
   - 胜率
   - 最大回撤

2. **执行质量**
   - 滑点 vs 收益
   - 延迟 vs 成功率
   - 被过滤比例

3. **系统稳定性**
   - 错误执行（必须 = 0）
   - 生命周期闭环率
   - 熔断触发次数

## 风险提示

- REST 缓存价格滞后 0-2 秒
- 100x 杠杆下价格偏差放大
- 不适合高频/突破策略
- 仅限低波动行情
