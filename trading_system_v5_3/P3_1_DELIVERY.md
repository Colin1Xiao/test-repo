## 三、技术实现

### 前端架构

**文件**: `history_analysis.html`  
**行数**: 320 行  
**依赖**: 无（纯原生 JS + CSS）

### 核心功能

```javascript
// 加载全部数据
async function loadAll() {
  await Promise.all([
    loadAlerts(),      // 告警
    loadControl(),     // 控制变更
    loadDecisions()    // 决策
  ]);
}

// 自动刷新（每 30 秒）
setInterval(loadAll, 30000);
```

### API 调用

```javascript
// 告警
fetch('/api/history/alerts?limit=20')

// 控制变更
fetch('/api/history/control?limit=20')

// 决策
fetch('/api/history/decisions?limit=1')
```
