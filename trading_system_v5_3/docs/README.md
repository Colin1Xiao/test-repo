## 📊 **评分标准**

### **趋势一致性 (30分)**
- EMA快线在慢线上方 → 做多趋势 → 30分
- EMA快线在慢线下方 → 做空趋势 → 30分
- 快慢线粘合 → 0分

### **回撤后突破 (20分)**
- 价格回踩快线后重新站回 → 20分
- 无回踩或未突破 → 0分

### **成交量确认 (15分)**
- 成交量 > 1.5x 均值 → 15分
- 成交量 > 1.2x 均值 → 10分
- 成交量 > 1.0x 均值 → 7分
- 成交量 ≤ 均值 → 0分

### **点差质量 (15分)**
- 点差 < 2.5 bps → 15分
- 点差 < 5 bps → 10分
- 点差 < 7.5 bps → 4分
- 点差 ≥ 7.5 bps → 0分

### **波动率适中 (10分)**
- 0.05% ≤ 波动率 ≤ 2% → 10分
- 波动率 < 0.05% → 3分
- 波动率 > 2% → 0分

### **RL过滤意见 (10分)**
- ALLOW → 10分
- REDUCE → 5分
- REJECT → 0分

## 📊 **影子模式评估器**

### **核心功能**
1. ✅ 接收 V4.1 同样的市场数据输入
2. ✅ 给出评分结果和是否达标
3. ✅ 记录候选信号统计，不参与真实交易

### **使用方法**

```python
from core.shadow_evaluator import ShadowModeEvaluator

# 创建评估器
evaluator = ShadowModeEvaluator()

# 评估市场数据
result = evaluator.evaluate(
    symbol='BTC/USDT:USDT',
    ohlcv_df=df,              # OHLCV数据
    rule_signal='BUY',        # V4.1 原规则信号
    spread_bps=3.5,           # 点差 (bps)
    volume_ratio=1.8,         # 成交量比率
    volatility=0.01,          # 波动率
    rl_decision='ALLOW'       # RL决策
)

# 查看统计报告
print(evaluator.get_stats_report())

# 导出统计数据
evaluator.export_stats()
```

### **记录的数据字段**

每条记录包含以下字段：

| 字段 | 说明 |
|------|------|
| `timestamp` | 评估时间 |
| `symbol` | 交易对 |
| `rule_signal` | V4.1 原规则信号 |
| `v42_total_score` | V4.2 总分 |
| `v42_score_breakdown` | 6个分项得分 |
| `v42_is_qualified` | 是否达标 (>=75) |
| `rl_opinion` | RL意见 |
| `market_state.price` | 当时价格 |
| `market_state.spread_bps` | 点差 |
| `market_state.volatility` | 波动率 |
| `market_state.volume_ratio` | 成交量状态 |

### **数据文件位置**

- **候选信号记录**: `data/shadow_signals.jsonl`
- **统计数据**: `data/shadow_stats.json`
