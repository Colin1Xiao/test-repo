## 🚀 三阶段上线计划

### Phase 1: 微资金验证 (当前)

**目标**: 验证"实盘 ≈ 模拟"是否成立

**配置**:
- 单笔仓位: 0.5%–1% 资金
- 目标样本: ≥100笔实盘
- 硬止损: -0.15%
- 环境过滤: ETH下午禁用 + 低成交量禁用

**通过标准**:
| 指标 | 要求 |
|------|------|
| 实盘均值 | ≥ +0.03% |
| 盈亏比 | ≥ 1.5 |
| 滑点影响 | < 30%收益 |

**新增保护**:
```python
if real_slippage > 0.05%:
    BLOCK_FOR_30_MIN  # 流动性不足的实时信号
```

## 🛡️ 执行层风控规则 (最终版)

### 必须
1. **硬止损**: -0.15% 无条件触发
2. **连续止损熔断**: 
   - 3次 → 暂停30分钟
   - 5次 → 暂停120分钟
3. **滑点保护**: >0.05% → 暂停30分钟

### 强制过滤
4. **ETH下午禁用**: 12-18时段禁止开仓
5. **ETH低成交量禁用**: <1.05x禁止开仓
