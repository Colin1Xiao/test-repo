# V5.4 关键安全补丁

**核心原则**: 不是"构建交易系统"，而是"消除不可控行为"

---

## 当前风险等级

| 维度 | 状态 | 说明 |
|------|------|------|
| Execution | 🟢 | 执行链真实 |
| Data | 🟢 | 数据链完整 |
| Control | 🟡 | 双执行入口风险 |
| Risk | 🔴 | 止损未完全托管 |

---

## 一、执行引擎禁用（必须"逻辑+物理双断路"）

### ❌ 不安全的做法

```python
if self.execution_engine:
    self.execution_engine.stop()
    self.execution_engine = None
```

**风险**: ExecutionEngine 线程可能还在跑，stop() 不一定同步停止

**后果**: 主线程用新执行链，旧线程还在偷偷下单

### ✅ 正确做法

```python
# 1. 全局硬开关（逻辑断路）
DISABLE_EXECUTION_ENGINE = True

# 2. 所有入口加检查
if DISABLE_EXECUTION_ENGINE:
    return None

# 3. 物理断路
self.execution_engine.stop()
self.execution_engine = None
```

**原则**: 逻辑断路 + 物理断路 = 双保险

---

## 二、Position Gate 必须双层

### ❌ 弱 Gate（当前）

只检查 `open_positions` 字典

**问题**: 交易所已有仓位 ≠ 本地状态

### ✅ 双层 Gate

```python
# 第一层：本地状态
if self.current_position is not None:
    print("⛔ Local Gate: 已有持仓")
    return None

# 第二层：交易所状态
if get_position_size(exchange, symbol) > 0:
    print("⛔ Exchange Gate: 已有持仓")
    return None
```

**必须两个都通过**

## 三、止损必须用真实成交价

### ❌ 错误做法

```python
stop_price = estimated_price * 0.995
```

**问题**: estimated_price ≠ 实际成交价

**后果**: 止损位置偏移，风险计算是假的

### ✅ 正确做法

```python
# 必须等待真实成交价
entry_price = result["execution_price"]  # 或 order["average"]
stop_price = entry_price * 0.995
```

---

## 四、止损订单完整参数（OKX 必需）

### ❌ 不完整参数

```python
params = {
    "stopPrice": stop_price,
    "reduceOnly": True
}
```

**风险**: 有概率 silently fail

### ✅ 完整参数

```python
params = {
    "stopPrice": stop_price,
    "reduceOnly": True,
    "tdMode": "cross",  # 或 isolated
}
```

---

## 五、Exit Source 必须加 Trigger Module

### ❌ 当前做法

```python
trade["exit_source"] = "STOP_LOSS"
```

**问题**: 不知道是谁触发的

### ✅ 正确做法

```python
trade["exit_source"] = "STOP_LOSS"
trade["trigger_module"] = "risk_controller"  # 谁触发的
```

**便于后续 debug**

## 六、修复顺序（严格按此执行）

### ✅ Step 1: 执行入口（最危险）

- 添加 DISABLE_EXECUTION_ENGINE 硬开关
- 所有入口加检查
- 停止并清空 ExecutionEngine

### ✅ Step 2: Position Gate（双层）

- 本地状态检查
- 交易所状态检查
- 两者都必须通过

### ✅ Step 3: 止损（用真实 entry）

- 等待真实成交价
- 计算止损价
- 挂止损单（完整参数）
- 记录 stop_order_id
