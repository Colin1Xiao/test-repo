## 二、修复优先级（工程级排序）

### 🔴 P0-1: Position Gate（防叠仓）

**风险**: 指数级放大

```python
def has_position(exchange, symbol):
    pos = exchange.fetch_positions([symbol])
    for p in pos:
        if abs(float(p.get("contracts", 0))) > 0:
            return True
    return False

# 执行前检查
if has_position(exchange, symbol):
    print("⛔ 已有持仓，跳过")
    return None
```

---

### 🔴 P0-2: Exit Source 记录（关键）

**风险**: 无法做 Edge 审计

```python
exit_source = "unknown"

if reason == "stop_loss":
    exit_source = "STOP_LOSS"
elif reason == "take_profit":
    exit_source = "TAKE_PROFIT"
elif reason == "time_exit":
    exit_source = "TIME_EXIT"
elif reason == "manual":
    exit_source = "MANUAL"

# 记录到交易
trade["exit_source"] = exit_source
```

**原因**: 这是一切分析的基础
